# Cafe Billing System

A simple cafe billing system built with Python and Tkinter.

## Features

- Calculate bills for multiple items
- Include tax and service charge calculations
- View price list
- Generate random order numbers
- Reset functionality

## Requirements

- Python 3.x
- Tkinter (usually comes with Python)

## How to Run

```bash
python cafe.py
```
